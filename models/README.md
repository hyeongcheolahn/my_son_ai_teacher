# 실제 포켓몬 3D 모델 슬롯

이 게임은 저작권상 **공식 포켓몬 3D 모델을 포함하지 않습니다.**
기본적으로는 코드가 만든 *스타일라이즈드(근사) 3D 캐릭터*를 사용합니다.

진짜 포켓몬 외형을 쓰고 싶으면, 직접 구한 `.glb`(glTF binary) 모델 파일을
**이 폴더에** 포켓몬 id 이름으로 넣으세요. 예:

```
models/pikachu.glb
models/charizard.glb
models/mewtwo.glb
```

- 파일 이름의 `id`는 게임 데이터(`src/data/regions.js`)의 포켓몬 `id`와 같아야 합니다.
- 파일이 있으면 그 포켓몬은 자동으로 진짜 모델로 렌더링되고, 없으면 절차적 모델로 폴백합니다.
- 모델은 정면을 +Z, 키 약 2 유닛 정도로 맞추면 가장 자연스럽습니다(코드에서 자동 스케일 보정 예정).

> 모델 파일은 가족 사적 이용 범위에서만 사용하세요. 배포/공유는 권장하지 않습니다.
